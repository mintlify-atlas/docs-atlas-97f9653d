---
title: "Get Library"
api: "GET /library"
description: "Retrieve all games in the authenticated user's library"
---

Returns a list of all games in the user's personal library, including both purchased games and games accessed through subscription.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  
  Example: `Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`
</ParamField>

## Access Types

The library supports two types of game access:

### Purchase Access
When `accessType` is `"purchase"`, the user has permanently purchased the game and retains access regardless of subscription status.

### Subscription Access
When `accessType` is `"subscription"`, the user has access to the game through an active subscription. Access may be revoked if the subscription expires or is cancelled.
