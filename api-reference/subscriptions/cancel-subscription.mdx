---
title: "Cancel Own Subscription"
api: "POST /sub/cancel"
description: "Cancel the authenticated user's active subscription"
---

Allows an authenticated user to cancel their own active subscription. This immediately deactivates the subscription and sets the end date to the current time, revoking access to subscription-based games.

## Authentication

Requires a valid JWT token in the Authorization header.

```bash
Authorization: Bearer <token>
```

## Request

This endpoint does not require a request body. The user to cancel is determined from the authentication token.

## Behavior

- Sets the subscription's `isActive` field to `false`
- Updates the `endDate` to the current timestamp
- User immediately loses access to subscription-based games
- Purchased games remain accessible
- Games added to library via subscription will no longer be accessible

## Impact on Game Access

When a subscription is cancelled:

<CardGroup cols={2}>
  <Card title="Lost Access" icon="ban">
    - Subscription-based games in library
    - Ability to add new subscription games
    - Subscription-exclusive content
  </Card>
  
  <Card title="Retained Access" icon="check">
    - Individually purchased games
    - Game save data and progress
    - User account and profile
  </Card>
</CardGroup>

## Example Scenario

A user subscribes to a monthly plan on March 5, 2026 at 10:00 AM:
- **Start Date**: 2026-03-05T10:00:00.000Z
- **Original End Date**: 2026-04-04T10:00:00.000Z
- **isActive**: true

The user cancels on March 5, 2026 at 2:30 PM:
- **Start Date**: 2026-03-05T10:00:00.000Z (unchanged)
- **New End Date**: 2026-03-05T14:30:00.000Z (current time)
- **isActive**: false

The user's subscription period lasted only 4.5 hours instead of the full 30 days.

## Important Notes

- Cancellation is immediate - no refunds are processed through this endpoint
- Users can subscribe again after cancellation
- To change plans, users must cancel current subscription first, then subscribe to a new plan
- This endpoint does NOT require admin privileges
