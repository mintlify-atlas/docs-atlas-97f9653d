## Welcome to GameHub API

GameHub API is a comprehensive RESTful platform designed for managing video games and user access through flexible subscription plans and purchase options. Built with Node.js and Express, it provides secure authentication, role-based access control, and a rich feature set for game library management.

## Architecture Overview

GameHub API follows a modern MVC architecture pattern:

- **Controllers** handle HTTP request/response logic
- **Models** define MongoDB schemas with Mongoose
- **Routes** organize endpoints by resource type
- **Middlewares** provide authentication, error handling, and validation
- **Helpers** contain reusable utility functions
