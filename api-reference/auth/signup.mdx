## POST /auth/signup

Creates a new user account and returns an authentication token.

### Authentication

No authentication required.

### Request Body

<ParamField body="name" type="string" required>
  The user's full name
</ParamField>

<ParamField body="email" type="string" required>
  The user's email address
</ParamField>

<ParamField body="password" type="string" required>
  The user's password
</ParamField>

<ParamField body="rol" type="string" default="user">
  The user's role. Can be `user` or `admin`
</ParamField>

### Response

<ResponseField name="status" type="string">
  Status of the response ("success")
</ResponseField>

<ResponseField name="token" type="string">
  JWT authentication token for the newly created user
</ResponseField>

### Example Request

```bash
curl -X POST https://api.gamehub.com/auth/signup \
  -H "Content-Type: application/json" \
  -d '{
    "name": "pedro luis",
    "email": "pedrorc2018@gmail.com",
    "password": "password123",
    "rol": "user"
  }'
```

### Example Response

```json
{
  "status": "success",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9"
}
```

### Error Responses

<ResponseField name="400" type="error">
  **Bad Request** - Missing email or password
  
  ```json
  {
    "status": "error",
    "message": "Por favor, ingrese correo y contraseña"
  }
  ```
</ResponseField>

<ResponseField name="500" type="error">
  **Internal Server Error** - Server error occurred
  
  ```json
  {
    "status": "error",
    "message": "Internal server error"
  }
  ```
</ResponseField>
