---
title: 'Get Game'
api: 'GET /game/{videojuegoId}'
description: 'Get detailed information about a specific game'
---

Returns detailed information about a specific video game. The amount of information returned depends on the user's access level:

- **Subscription-only games**: Full details only for users with active monthly/annual subscription
- **Purchase-only games**: Full details only for users who have purchased the game
- **Both subscription and purchase**: Full details for subscribers or purchasers

## Authentication

Requires JWT authentication token in the Authorization header:

```
Authorization: Bearer <token>
```

## Path Parameters

<ParamField path="videojuegoId" type="string" required>
  The unique identifier of the video game
</ParamField>

## Access Control Logic

Implemented in `src/controllers/videojuego.controller.js:33-117`:

### Subscription-Only Games
- `accessType.subscription = true` and `accessType.purchase = false`
- **Full access**: Users with monthly or annual subscription
- **No access**: Users without subscription (403 error)

### Purchase-Only Games
- `accessType.subscription = false` and `accessType.purchase = true`
- **Full access**: Users who have purchased the game
- **Limited access**: Other users see title, genre, release date, and price

### Dual Access Games
- `accessType.subscription = true` and `accessType.purchase = true`
- **Full access**: Subscribers OR purchasers
- **Limited access**: Non-subscribers who haven't purchased
